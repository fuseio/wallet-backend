# Wallet Backend API

## Description
A boilerplate backend infrastructure for the Fuse wallet.

## Installation

```bash
$ yarn install
```

## Setup environment variables
```bash
$ cp .example.env .env
```

## Running the apps

```bash
# development in watch mode
$ yarn docker:debug

# production WIP
$ yarn docker:staging
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
