# Wallet Backend API

## Description
The voltage backend api 

## Installation

```bash
$ yarn install
```

## Setup environment variables
```bash
$ cp .example.env .env
```

## Running the apps

```bash
# development in watch mode
$ npm run docker:debug

# production WIP
$ npm run docker:staging
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
